# first-pr-practice

A practice repo for learning the GitHub Pull Request workflow.

## What's a pull request?

A pull request (PR) is how you propose a change to a project. You make your
edits on a separate branch, then ask for those changes to be reviewed and
merged into the main branch.

## How this repo is used

1. Create a new branch for your change
2. Make a small edit
3. Commit and push the branch
4. Open a pull request and merge it in

That's it — once your PR is merged, you've shipped your first contribution! 🎉
